💎 Section 2: Gems (役割の固定)

🤵 Gems設定テンプレート

「役割」と「手順」をGeminiに記憶させるためのテンプレートです。

# あなたの役割 # Role あなたはプロフェッショナルなエグゼクティブ秘書、兼プロジェクトマネージャーです。 提供された「議事メモ」や「音声スクリプト」を解析し、構造化された正確な議事録を作成してください。 # Task ユーザーが入力したテキストから、以下のフォーマットに従って議事録を生成してください。 情報の不足がある場合は、推測せず「不明」とするか、項目自体を省略してください。 # Output Format ## 1. 会議概要 - **議題:**（会議のメインタイトル） - **日時:** - **参加者:**（敬称略） ## 2. 決定事項 - [ ] （決定した内容を箇条書きで記載） ## 3. 議事詳細（要約） ### [アジェンダ1] - （議論の要点を簡潔に記載） - （重要な発言や背景があれば補足） ### [アジェンダ2] - ... ## 4. ネクストアクション（宿題） | タスク内容 | 担当者 | 期限 | | :--- | :--- | :--- | | 例：資料の修正 | 山田 | 2/20 | ## 5. 次回予定 - （次回のミーティング日時や持ち越し事項） # Guidelines - **簡潔さ:** 冗長な表現は避け、箇条書きを活用して「一目でわかる」構成にしてください。 - **正確性:** 発言の意図を汲み取りつつ、事実に忠実にまとめてください。 - **フィルタリング:** 「えーと」「あのー」などのフィラーや、本筋に関係のない雑談はカットしてください。 - **トーン:** 丁寧でプロフェッショナルなビジネス敬語を使用してください。

📝 1-2. テスト用データ：議事録サンプル

作成したツールの動作確認に、以下のテキストを使用してください。

佐藤： お疲れ様、高橋君。忙しいところ悪いね。今日は例のキャリア面談ということで、今の仕事の状況と、今後のステップアップについて少し腹を割って話せればと思ってるんだ。最近はどうかな？ 高橋： お疲れ様です。そうですね……。担当している自動車部品メーカー向けの既存案件は安定しているんですけど、正直なところ、最近は「ただの御用聞き」になっている気がして。見積もりを出して、納期を調整して、たまにクレーム対応して……というサイクルの繰り返しで、営業としての成長が止まっているんじゃないかと不安になることがあります。 佐藤： なるほど。既存顧客の維持も立派な仕事だけど、高橋君としてはもっと付加価値の高い提案をしたい、ということかな？ 高橋： はい。実は、今の製品ラインナップに加えて、最近トレンドになっている「環境配慮型の新素材部品」をもっと積極的に提案していきたいんです。でも、今の業務フローだと見積作成や納期の催促といった事務作業に追われていて、新しい技術資料を読み込んだり、顧客の設計部門に食い込んだりする時間が全然取れなくて。 佐藤： 事務作業の負担か。それはチーム全体の課題でもあるな。あ、そういえばちょっと話逸れるけど、先週の展示会で使ったパネルの保管場所、ちゃんと倉庫のB棟に移しておいてくれた？ 邪魔だって資材部から小言言われちゃってさ（笑）。 高橋： あ、すみません！ 戻し忘れてました。今日、外回りから帰ったらすぐにやっておきます。 佐藤： 悪いね。で、本題に戻るけど、高橋君は「ソリューション型」の営業にシフトしたいってことだよね。単にモノを売るだけじゃなくて、顧客のカーボンニュートラル対応まで踏み込んだ提案をしたいと。 高橋： おっしゃる通りです。ただ、今のメイン担当であるA社（大手メーカー）のボリュームが大きすぎて、どうしてもそこを優先せざるを得ないんです。このままだと5年後、10年後も同じことをしている気がして。 佐藤： 分かった。高橋君の意欲はよく伝わったよ。だったら、次の下半期から少し役割を変えてみよう。今のA社の定型業務の一部を、新人の鈴木君に引き継ぐのはどうだろう？ 彼は彼で、実務を覚える段階だからちょうどいい。 高橋： 鈴木君にですか。確かに、彼もだいぶ慣れてきましたし、私のサポートがあれば回せるかもしれません。 佐藤： よし。じゃあ条件だ。2週間以内に「新素材部品のターゲット顧客リスト」と「初期提案用のプレゼン骨子」を作ってみてくれないか？ それを営業会議で通せれば、正式に高橋君を「新規事業開発兼務」としてアサインするよ。 高橋： 本当ですか！ ありがとうございます。やらせてください。リストと骨子ですね。3月最初の金曜日までには仕上げます。 佐藤： 期待してるよ。それと、その「環境配慮型」の提案をするには、少し専門的な知識も必要だよね。会社で推奨している「環境経営士」の資格とか、興味ある？ 取得費用は会社が持つ制度があるんだ。 高橋： あ、それ気になっていました。試験は6月ですよね。今から勉強を始めれば間に合うと思います。ぜひ活用させてください。 佐藤： よし、それは人事に確認して、後で申請マニュアルを共有するよ。あと、さっき言った「事務作業の効率化」についても、何かアイデアがあれば教えてほしい。高橋君が楽になれば、他のメンバーも助かるはずだから。 高橋： そうですね……。例えば、見積依頼のフォーマットを共通化するだけでも、設計部門とのやり取りがかなりスムーズになる気がします。今まで電話やメールでバラバラだったので。 佐藤： それは良い着眼点だ。その見積フォーマットのたたき台も、併せて考えておいてくれるかな？ 高橋： 承知しました。新規提案の準備、資格の勉強、それに見積フォーマットの改善ですね。 佐藤： うん。あとは、新人の鈴木君の指導もよろしく頼むよ。彼、高橋君の営業スタイルを尊敬してるみたいだから。週に一度、移動中とかでもいいから、彼と状況確認する時間を取ってあげて。 高橋： わかりました。鈴木君との定期的な打ち合わせもスケジュールに入れます。 佐藤： よし。まとめると……まずは倉庫のパネル移動（笑）。それから2週間後の提案準備。資格の申請。見積フォーマットの検討。そして鈴木君のメンター。盛りだくさんだけど、大丈夫かな？ 高橋： はい！ 自分のやりたかった方向に進めるチャンスなので、全力でやります。 佐藤： その意気だ。期待しているよ。あ、いけない、次の来客が来ちゃった。今日はここまで。 高橋： ありがとうございました！ 失礼します。

⚙️ Section 3: GAS (業務自動化)

📂 ファイル名一括変更スクリプト

⚠️ 実行する際は必ず「練習用の空フォルダ」を作って試してください。

# 命令 Google Apps Script (GAS) のコードを書いてください。 # やりたいこと Googleドライブの「特定のフォルダ」に入っているすべてのファイルに対して、ファイル名の先頭に「今日の日付（YYYYMMDD_形式）」を追加したいです。 # 要件 1. フォルダIDは、スクリプトの冒頭で変数として指定できるようにする。 2. 重複実行防止機能をつける（既に日付がある場合はスキップ）。 3. 実行完了時にログを出力する。
